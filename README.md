# PEDROGAMES

Site inicial do projeto PEDROGAMES.

Quando o GitHub Pages estiver ativado, o endereco esperado sera:

https://rsscavalcanti-cmyk.github.io/PEDROGAMES/

## Como publicar pelo GitHub Pages

1. Abra o repositorio no GitHub.
2. Entre em Settings > Pages.
3. Em Build and deployment, escolha Deploy from a branch.
4. Selecione a branch `main` e a pasta `/root`.
5. Salve. O GitHub vai gerar o link publico do site.

## Arquivos

- `index.html`: estrutura da pagina.
- `styles.css`: visual do site.
- `script.js`: pequenas interacoes.
